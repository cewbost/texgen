loadModel

function loadModel(filename)

Loads a model resource from disk and sets it as the current model for the renderer.  The model resource format is Horde3Ds *.geo format.

setModel

function setModel(model)

Sets the current model resource for the renderer.

parameters

modela blob containing a binary Horde3D *.geo resource, or a table containing multiple blobs.

If the parameter is a table, the table should contain the following slots

verticesa blob containing the models vertices as 4-byte float triplets.
indicesa blob containing the models vertex indices stored as three 4-byte integers per triangle.
normalsa blob containing the models normals as 4-byte float triplets(optional).
tangentsa blob containing the models tangents as 4-byte float triplets(optional).
binormalsa blob containing the models binormals as 4-byte float triplets(optional).
texcoords1a blob containing the first set of texture coordinates for the model stored as 4-byte float pairs(optional).
texcoords2a blob containing the seconds set of texture coordinates for the model stored as 4-byte float pairs(optional).
tanspace(bool) if true, then the binormals or tangents will be generated automatically(optional)(normals and either tangents or binormals must be supplied).
